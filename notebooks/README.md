# Notebooks

This directory contains Jupyter notebooks for the CrediTrust RAG Complaint Chatbot project.

## Contents

| Notebook | Task | Description |
|----------|------|-------------|
| `task1_eda.ipynb` | Task 1 | Exploratory Data Analysis and Preprocessing of CFPB complaint data |
| `task3_evaluation.ipynb` | Task 3 | RAG Pipeline Qualitative Evaluation — 8 test questions with quality scoring |

## Running the Notebooks

```bash
# From the project root
jupyter notebook notebooks/
```

Or open individually in VS Code with the Jupyter extension installed.

## Requirements

All notebook dependencies are in the project `requirements.txt`. Install with:
```bash
pip install -r requirements.txt
```
